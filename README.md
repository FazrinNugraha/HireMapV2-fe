# HireVision Frontend

React + Vite frontend untuk HireVision FastAPI backend.

## Environment

Salin `.env.example` menjadi `.env` jika ingin mengganti URL backend:

```env
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## Scripts

```bash
npm run dev
npm run build
```

Pastikan backend berjalan di URL yang sama dengan `VITE_API_BASE_URL`.
